# MLFlow

### MLOps

- 머신러닝 + 데브옵스
- 머신러닝 워크플로의 자동화, 실행

### Key Components of MLOps

- Model Deployment
  - .pkl, .onnx
  - Deploy Model(Rest API, Docker, FastAPI)
- Data Management
  - Dataset handling
  - Airflow, Kafka
- Model Monitoring
  - Prometheus, Grafana
- Model Development
  - 모델 버저닝
  - 하이퍼파리미터 튜닝
- CI/CD
  - Github Actions
- Governance and Compliance
